## draft 简介

draft 是一个帮助开发者在 kubernetes 上创建云原生应用的工具, 简化了应用开发和部署的过程.

这是我在 vscode 上安装 kubernetes 相关插件时看到的.

最近好像没有更新, 还是停留在 18 年的 0.16 版本.

## 安装

去 github 上的 [release 页面](https://github.com/Azure/draft/releases) 下载二进制文件.

```bash
# 初始化
draft init

# 创建应用
draft create

# 部署应用
draft up
```
